The directory for PasDoc scripts
------------------------------------------------------------
To generate GBSysHelp.chm you need to follow the next steps:
1.Download PasDoc program from http://pasdoc.sourceforge.net/ and setup it.
2.Run the pasdoc_gui program and Open GBSysHelp.pds script 
3.Click on Generate button and create system help files in HTML folder
4.Download HTML Help Workshop 1.3 software from Microsoft site and setup it.
5.Run the hhw.exe program from C:\Program Files (x86)\HTML Help Workshop\
6.Open GBSysHelp.hhp project in ..\en\GBSysHelp\HTML\ folder and compile it.
7.Clean HTML folder using _CleanHTML.bat and find there GBSysHelp.chm
