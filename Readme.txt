The release of Gexoblock v. 2020.04.01 alpha
Compiled using Embarcadero RAD Studio for Delphi/C++Builder 

The Gexoblock is an integrated software for 
computational geometry and visualization of spatial datasets. 
The program can be used for Earth sciences and particularly for geological modelling, 
ore reserve estimations and open pit mining, prediction of mineral liberation 
under grinding and mineral processing operations.
